Thayegu
is a Django-based web application built with Python, HTML, and CSS. It is designed as a learning and experimentation project to explore backend logic, database integration, and frontend rendering in a clean, structured way.

Features
- Django Framework: Provides a robust backend structure.
- SQL Database: Lightweight and easy to set up (db.sqlite3 included).
- Frontend Integration: HTML and CSS for user interface design.
- Manage.py Utilities: Standard Django commands for migrations, running the server, and app management.

Tech:

Backend:Python(Django)

Database:Mysql

Frontend:HTML,CSS,JavaScript

Prerequisites

Make sure you have the following installed:
- Python 3.8+
- pip (Python package manager)
- virtualenv (recommended)


Clone the repository


git clone https://github.com/GodANDDevil/Thayegu.git

cd Thayegu

- Create a virtual environment

python -m venv venv
source venv/bin/activate   # On Linux/Mac
venv\\Scripts\\activate    # On Windows

Install dependencies

pip install -r requirements.txt

Project Structure

Thayegu/
│── Thayeguu/        # Main Django app
│── db.sqlite3       # Database file
│── manage.py        # Django project manager
